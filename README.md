# Rabbit-Run---Hopp-oder-Loch
Dieses Repository enthält meine Java-Umsetzung eines bekannten Brettspiel. Ziel war es, die Spielmechanik rund um sich öffnende Bodenfelder und das Wettrennen zum Gipfel softwaretechnisch abzubilden. Es war eines meiner ersten größeren Projekte, bei dem ich mich intensiv mit objektorientierter Programmierung und GUI in Java auseinandergesetzt habe.
